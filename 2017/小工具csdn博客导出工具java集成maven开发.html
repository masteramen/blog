CSDNBlogExport

CSDN博客导出工具

之前一直想把CSDN的博客导入到自己的网站中，可是由于博客比较多，后面受朋友老郭启发，就找了个时间用Java开发了这款小工具。 Had been trying to CSDN blog into their website, but because of the blog is more, inspired by my friend guo behind, will find a time this kind of small tools with Java development.

only use

直接下载CSDNBlogExport.7z解压使用即可。 Direct download CSDNBlogExport.7z decompression can be used.

经过测试，667篇博客，开50个线程，在54秒左右可以全部导出到文件。 Tested, 667 blog, open 50 threads, in 54 seconds can all exported to a file.

博客文件导出的存储规则是: 软件运行目录\blog\年-月\年-月-日 博客标题名.markdown Blog file exported storage rule is: Software running directory/blog/year-month/year-month-day blog title name.markdown
