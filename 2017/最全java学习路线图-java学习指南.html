1.基本的语法规范 2.数据类型 3.流程控制 4.函数

1.JavaOOP之封装 2.JavaOOP之继承 3.JavaOOP之多态 4.Java访问修饰符 5.Java接口 6.Java抽象类

●集合框架和数据结构

1.Java常用集合 2.Java泛型

●异常处理

1.异常分类、原因、概述、原理、解决方案 2.LOG4J框架
